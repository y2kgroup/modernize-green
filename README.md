# Modernize Green Solutions — React Site

Migrated from WordPress (Avada theme) to Next.js + Tailwind.

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build for production

```bash
npm run build
```

Static files will be in `./out/`. Vercel will run this automatically when you push to GitHub.

## Editing content

All site content lives in `site.config.json`. Change text, swap images, update navigation — no need to touch the code.
